 - `docker container ps` shows all the running images
 
 - Use ` docker compose up -d ` for everything up and running
 
 - Use `docker compose stop` to stop the running container .. frees the RAM no network removed easy resume
 
 - Use 'docker compose start' after docker compose stop to resume the container running in the docker .
 
 - Use 'docker compose down' to remove the container , remove the container, remove the network it removes everything.
