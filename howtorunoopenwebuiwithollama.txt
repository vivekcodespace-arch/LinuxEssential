

1. Create network(if not created)
docker network create ollama-net

(ignore error if already exists)


Step 2: use this to run them on the same port:

- docker network connect ollama-net ollama

Step 3: Run Ollama
docker run -d \
  --network ollama-net \
  -v ollama:/root/.ollama \
  -p 11434:11434 \
  --name ollama \
  ollama/ollama


Step 4: Run OpenWebUI (FULL BACKEND MODE)
docker run -d \
  --network ollama-net \
  -p 3000:8080 \
  -e OLLAMA_BASE_URL=http://ollama:11434 \
  -e WEBUI_AUTH=False \
  --name open-webui \
  ghcr.io/open-webui/open-webui:main
  
 Step5:  After setting it once
	- You just need to use docker start ollama open-webui
	- Check for the running status using sudo systemctl status ollama, because ollama may be used at the system level
